Requisitos de UNGenético 2.0

UNGenético es una librería multiplataforma desarrollada en lenguaje C++, esta librería ha sido probada en los sistemas operativos Windows y Linux. Los requisitos mínimos para su funcionamiento son:

Sistema operativo Windows

  • Compilador del lenguaje C++
    Recomendado: Visual C++ 6.0
  • Librería wxWindows
    Recomendado: wxWindows 2.4

Sistema operativo Linux

  • Compilador del lenguaje C++
    Recomendado: gcc
  • Librería wxGTK
    Recomendado: wxGTK 2.4

NOTA: La creación de proyectos de wxWindows puede generar problemas en algunos compiladores, por lo tanto, es recomendable utilizar las aplicaciones diseñadas para generar proyectos de este tipo según el compilador de C++ donde se construya el proyecto. Estas aplicaciones se pueden descargar del sitio oficial de wxwindows www.wxwindows.org.
